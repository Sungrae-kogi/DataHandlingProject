# 빅데이터 처리 프로젝트

##  1. 프로젝트 소개
#### 89GB 텍스트 파일을 주어진 목표에 따라 처리하는 프로그램 제작.
- **개발 기간**: 2026-01-07 ~ 2026-01-14(1차 성능개선 완료) ~ 진행중

##  2. 프로젝트 핵심 목표
-  **데이터 갯수 파악**
-  **각 데이터 컬럼별 최대 데이터 길이 파악**
-  **출원일자의 년도를 읽고 같은 년도끼리 데이터 파일 분리**
##  3. KEY SUMMARY
ChatGpt에 이론을 제외한 직접적인 코딩요청 X , 최대한 Java 문서(핵심)와 구글링을 사용하여 직접 구현 및 개선 진행.
